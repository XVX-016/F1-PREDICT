## Documentation Standards

All new `.md` files should be created in this `docs/` folder only.

To create new documentation:

```bash
# Create new documentation file
touch docs/NEW_DOCUMENTATION.md
# or
echo "# New Documentation" > docs/NEW_DOCUMENTATION.md
```
