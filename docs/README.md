## Documentation Structure

All markdown (.md) files should be placed in this `docs/` folder for better organization.

## Creating New Documentation

When creating new .md files, place them in this `docs/` folder:

```bash
# Example: Create new documentation
touch docs/NEW_FEATURE.md
# or
echo "# New Feature" > docs/NEW_FEATURE.md
```

This ensures all documentation is centralized and easy to find.
